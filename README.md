# Jeti_VarioMeter
Jeti VarioMeter

Code for Jeti Variometer

Simple Variometer sensor - Vertical speed in m/s or ft/s

 Includes
 - Altitude
 - Temperatue
 - Pressure
 - Vario

 Localization selectable vith Jetibox by user to EU / US:
 - Altitude     m / ft
 - Temperature  C / F
 - Pressure     hPa / inHG
 - Vario        m/s / ft/s

More info: https://www.rc-thoughts.com/jeti-cdi-sensor/
